## 一、简介

该插件用于在玩家登录后将玩家送到最后游玩的子服

依赖MYSQL，支持识别AuthMe的登录

## 二、插件文档

[链接](https://plugin.sandtripper.cn/SandBackLastServer/)

## 三、安装插件

直接将插件文件拖入plugins，可选择重启服务端，也可热加载

每个子服都需要安装，并配置好数据库和is-lobby选项
